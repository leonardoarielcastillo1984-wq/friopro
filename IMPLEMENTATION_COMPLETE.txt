🔒 RATE LIMITING

  │  ├─ Password Reset: 3 attempts/hour
  │  ├─ API Endpoints: 100 requests/minute

  ├─ Response Headers: X-RateLimit-*
  ├─ Error Status: HTTP 429 when exceeded

================================================================================
                            TROUBLESHOOTING GUIDE
================================================================================

EMAILS NOT SENDING:
  1. Check Redis: redis-cli PING
  2. Check credentials: echo $RESEND_API_KEY
  3. Check logs: grep -i "EmailWorker" /var/log/sgi360/*.log
  4. See: EMAIL_SETUP_GUIDE.md → Troubleshooting

AUDIT LOGS NOT RECORDING:
  1. Check DB: psql $DATABASE_URL -c "SELECT COUNT(*) FROM \"AuditEvent\";"
  2. Check logs: grep -i "AuditLog" /var/log/sgi360/*.log
  3. See: AUDIT_LOGGING_GUIDE.md → Troubleshooting

RATE LIMITS NOT WORKING:
  1. Check Redis: redis-cli KEYS 'ratelimit:*'
  2. Check middleware: Verify preHandler registered
  3. See: RATE_LIMITING_GUIDE.md → Troubleshooting
