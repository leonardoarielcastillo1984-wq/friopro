LOGIN FLOW (New):
-----------------

1. User enters email and password
2. Backend verifies credentials
3. Backend skips 2FA check
4. Backend generates access token + refresh token
5. User receives tokens and is logged in
6. User is shown success page with option to "Go to Dashboard"
7. User can optionally enable 2FA in account settings later


KEY FEATURES:
--------------

✓ Simplified login: Email + Password only
✓ No required 2FA on login
✓ Direct portal access after credentials
✓ 2FA available as optional setting (can be enabled later by users)
✓ Maintains existing security: password hashing with argon2
✓ Maintains existing token management
