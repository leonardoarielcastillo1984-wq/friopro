## 🚀 Cómo Usar

### 1. **Descomprimir**
- Buscá `SGI-360-Launcher-App.zip`
- Hacé doble clic para descomprimir

### 2. **Ejecutar**
- Hacé doble clic en `sgi360-launcher.app`
- Aparecerá un diálogo con opciones

---

## 🎮 Interfaz Gráfica

### **Diálogo Principal:**
```
SGI 360 - Sistema de Gestión Integrado

¿Qué querés hacer?

[ Iniciar Todo ] [ Detener Todo ] [ Cancelar ]
```

### **Al iniciar:**
- 📢 Notificaciones de progreso
- 🌐 Abre navegador automáticamente
- ✅ Mensaje de éxito con datos de acceso

### **Si hay error:**
- ❌ Mensaje claro del problema
- 📋 Ubicación de logs para revisar

---

## 🌐 Funcionalidades

### **Iniciar Todo:**
- Verifica Docker
- Libera puertos
- Inicia PostgreSQL + Redis
- Configura base de datos
- Carga datos de demo
- Inicia API y Web
- Abre navegador

### **Detener Todo:**
- Detiene servicios
- Libera puertos
- Cierra contenedores Docker
